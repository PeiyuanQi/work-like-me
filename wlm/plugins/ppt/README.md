# PPT Plugin

Presentation style adapters and third-party PPT skill references.

## Styles

- `ppt-style-selector`: choose among registered PPT styles before generating a
  deck.
- `ppt-guizang-style`: adapter for `op7418/guizang-ppt-skill`. Use only when
  the user asks for the Guizang PPT style, its electronic magazine/e-ink visual
  direction, its Swiss international style direction, or explicitly chooses this
  style from the PPT style set.

## Policy

- Treat each PPT style as one selectable style, not as the universal
  presentation workflow.
- Prefer referencing or wrapping third-party presentation skills over copying
  them into work-like-me.
- Do not vendor third-party skill content unless the license is verified and
  required notices are recorded.
