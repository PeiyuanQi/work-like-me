# Mechanical Plugin

Mechanical and hands-on build skills for physical projects, repairs, regulated
devices, tools, materials, and practical engineering tradeoffs.

## Skills

- `rank-mechanical-decisions`: rank options for physical builds, repairs,
  device choices, materials, tooling, safety, compliance, and maintenance.

## Workflow Preferences

- Put safety, legality, licensing, and irreversible physical damage first.
- Prefer proven off-the-shelf parts and small tests before custom fabrication.
- Treat real materials, tolerances, tools, and operator skill as constraints.
- Document failures and practical workarounds so the next attempt is cheaper.
