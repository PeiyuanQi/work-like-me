# Worker Plugin

Daily office worker skills for productivity, memory, team collaboration, and
presentation workflows.

## Skills

- `consolidate-memory`: find and normalize the canonical memory location.
- `contact-new-teammate`: draft a polite first contact to a coworker.
- `find-poc`: find a person of contact for non-code-review requests.
- `notify-reviewer`: notify the selected reviewer about an MR or PR.
- `onboard-as-new-hire`: discover and record team tools and org context.
- `search-memory`: search stored memory for team or project context.
- `update-memory`: save useful team or project context to memory.
- `rank-office-decisions`: rank normal office, productivity, career,
  communication, and prioritization choices with an anonymized blog-derived
  value pattern.
- `ppt-style-selector`: choose among registered PPT styles before generating a
  deck.
- `ppt-guizang-style`: wrapper for `op7418/guizang-ppt-skill`; use only when
  Guizang is explicitly selected or is the intended work-like-me PPT style.

## Presentation Policy

- Treat PPT work as normal office-worker productivity.
- Treat each PPT visual direction as one selectable style, not as the universal
  presentation workflow.
- Prefer referencing or wrapping third-party presentation skills over copying
  them into work-like-me.
- Do not vendor third-party skill content unless the license is verified and
  required notices are recorded.

## Decision Policy

- Treat normal office decisions as agency, time, information, relationship, and
  option-preservation tradeoffs.
- Prefer useful evidence and one concrete next step over status games, process
  theater, or abstract moralizing.
